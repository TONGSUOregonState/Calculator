Block 公式 Demo：v, a = v/t, d = 0.5·a·t²

左边是输入 Block，右边是计算 Block。你可以选择每个 input 的来源（上游 Block 或手动值）。
这是"Block + 输入输出 + 数据流"的逻辑 Demo，将来可以换成拖拽连线 UI。
Block 1：速度 v
输入 v，然后作为 output 提供给其它 Block
Output: v = {{ pretty(vInput) }}
Block 2：时间 t
输入 t，然后作为 output 提供给其它 Block
Output: t = {{ pretty(tInput) }}
Block 3：加速度 a
公式：a = v / t
使用的 v = {{ pretty(aV) }}，t = {{ pretty(aT) }}
a = v / t = {{ isFinite(aValue) ? pretty(aValue) : 'NaN (t 不能为 0)' }}
Block 4：位移 d
公式：d = 0.5 · a · t²
使用的 a = {{ pretty(dA) }}，t = {{ pretty(dT) }}
d = 0.5 · a · t² = {{ pretty(dValue) }}
（当 d 的 a 来源选 "Block 3"，就相当于把上一个 Block 的 output 拖到这里的 input）
